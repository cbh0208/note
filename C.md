### 位操作

```c
// 把整数中的第n位置1
a = a | ( 1 << n );

// 把整数中的第n位置0
a = a &( ~( 1 << n) );

// 把整数中的第n位反转
a = a ^ ( 1 << n );

// 测试整数中的第n位是否为1（0）
b = a & ( 1 << n );	// b的值不等于0说明被测整数的第n位是1，b的值是0说明被测整数的第n位是0。

```
